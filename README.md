Two-Digit Decimal Counter (0–99) using 74LS192 and 7447 ICs implemented in Proteus.

This project is a Digital Electronics simulation that demonstrates how to design a synchronous BCD counter system capable of counting from 00 to 99 using TTL integrated circuits.

The circuit uses:
- 74LS192 BCD Up/Down Counters
- 7447 Seven-Segment Decoder Drivers
- Seven-Segment Displays
- Logic Gates
- Buzzer Alarm Circuit

✨ Features:
- Counts from 00 to 99
- Real-time display using seven-segment displays
- Cascaded counter design
- Alarm/Buzzer indication
- Proteus simulation ready

📚 Concepts Covered:
- BCD Counting
- Seven-Segment Display Interfacing
- Cascading Counters
- Digital Logic Gates
- Synchronous Counter Operation

🛠 Software Used:
- Proteus Design Suite

This project was created as part of the Digital Electronics course to apply practical digital logic design concepts.
